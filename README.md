# Credit EDA Case Study

This repository contains an Exploratory Data Analysis (EDA) notebook for a credit dataset.  
The notebook was originally created and run on Kaggle, and is now linked here.

## How to Run

### Option 1: Run Directly on Kaggle
- The notebook is set up to run on Kaggle with the embedded Kaggle dataset path.
- You can view and execute the notebook on Kaggle without any changes.

### Option 2: Run Locally
- **Download the dataset** from Kaggle to your local machine.  
  [Dataset link](https://www.kaggle.com/datasets/venkatasubramanian/credit-eda-case-study)
- **Update the data loading code** in the notebook:
  - Replace the Kaggle dataset path (e.g., `/kaggle/input/credit-eda-case-study/application_data.csv`) with your local path (e.g., `./application_data.csv`).
- **Install required libraries** (see below).

## Required Libraries

- pandas
- numpy
- matplotlib
- seaborn
